# ⚙️ Local Agents – Frontend (app/)
## 🎯 Goal
Offline-first UI using **Next.js 15**, **Tailwind CSS v4**, and **TypeScript**.

## 👥 Local Agents

### ⚙️ UXAgent

- Form validation & toasts.

Dependencies: next@15, tailwindcss@4, react-hook-form, axios, next-auth.
